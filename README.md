# Diabetes Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a Diabetes dataset to identify patterns and key health indicators.

## Objectives
- Understand distribution of health parameters
- Identify correlations between features
- Analyze factors affecting diabetes outcome

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning
- Statistical summary
- Correlation analysis
- Data visualization (histograms, bar charts, heatmap)

## Conclusion
The analysis shows relationships between glucose levels, BMI, and diabetes outcome. Further predictive modeling can be performed in future.
